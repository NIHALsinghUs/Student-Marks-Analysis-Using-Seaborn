# Student Marks Analysis Using Seaborn

## Overview

Student Marks Analysis Using Seaborn is a beginner-friendly Python project designed to demonstrate data analysis and visualization techniques. The application allows users to enter student details and subject marks, stores the data in a Pandas DataFrame, calculates average marks, and generates meaningful visualizations using Seaborn and Matplotlib.

## Features

* Add student records dynamically through user input.
* Store student data in a Pandas DataFrame.
* Save records to a CSV file.
* Calculate average marks automatically.
* Generate Bar Plots for student performance comparison.
* Generate Histograms to analyze marks distribution.
* Generate Heatmaps to visualize subject correlations.
* Menu-driven interface for easy interaction.

## Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib

## Visualizations

### Bar Plot

Displays the average marks of each student, making it easy to compare overall performance.

### Histogram

Shows the distribution of average marks and helps identify trends in student performance.

### Heatmap

Visualizes correlations between Maths, Science, English, and Average marks.

## Learning Outcomes

This project helps beginners understand:

* Data collection and storage using Pandas.
* Data manipulation and analysis.
* Exploratory Data Analysis (EDA).
* Data visualization with Seaborn.
* Working with CSV files in Python.

## Project Structure

* Student data input system
* Average marks calculation
* CSV file generation
* Visualization menu
* Bar Plot, Histogram, and Heatmap generation

## Conclusion

This project is an excellent starting point for learning data analysis and visualization in Python. It combines data collection, processing, and graphical representation into a single application, making it suitable for students, beginners, and portfolio projects.
